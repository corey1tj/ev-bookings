# ev-bookings

Public booking UI and admin console for Future Energy's EV charging network. Drivers browse charging locations, pick a time slot and charger, then confirm a booking.

Built with Next.js 14, TypeScript, and Tailwind CSS on top of the Ampeco Bookings API.

## Getting Started

All application code lives in the `fe-bookings/` directory. See [fe-bookings/README.md](fe-bookings/README.md) for setup instructions, environment variables, and detailed documentation.
